# 💳 Responsive Pricing Panel

A responsive pricing panel built using **HTML5** and **CSS3** as part of my frontend web development learning journey.

This project focuses on building a clean and modern pricing section while practicing Flexbox, responsive design, media queries, transitions, and CSS styling techniques.

---

## 🚀 Features

- Responsive layout using Flexbox
- Mobile-friendly design with Media Queries
- Three pricing plans (Personal, Small Team, Enterprise)
- Smooth button hover effects
- Modern card layout with rounded corners
- Clean and minimal UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- Media Queries

---

## 📚 Concepts Practiced

- CSS Box Model
- Flexbox Layout
- Responsive Design
- Media Queries
- CSS Transitions
- Hover Effects
- Typography
- Borders & Border Radius
- Spacing using Margin & Padding

---

## 📸 Preview

Responsive pricing table with three pricing plans and a modern card-based layout.

---

## 🎯 Learning Outcome

Through this project, I gained hands-on experience in:

- Creating responsive layouts
- Building reusable UI components
- Using Flexbox for alignment and spacing
- Writing clean and organized CSS
- Understanding how media queries adapt layouts for different screen sizes
